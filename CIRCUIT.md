## Connections

- Smoke sensor `OUT -> A0`, `VCC -> 5V`, `GND -> GND`.
- Buzzer `+ -> D9` (with driver/transistor if needed), `- -> GND`.
- LED `-> D13` through resistor.

```

 5V/GND         D13 -> [330Ω] -> LED -> GND
```
